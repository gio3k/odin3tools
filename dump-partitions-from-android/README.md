# Dumping Partitions

1. Download the [dumper script](https://raw.githubusercontent.com/gio3k/odin3tools/refs/heads/main/dump-partitions-from-android/dump.sh) (right click > Save Page As...)
2. Put the dumper script somewhere on your Odin 3's storage
3. Go to Android settings > Odin settings > Run script as Root > Select a script
4. Select the dumper script and press Run
5. Your dump should be in the Dumps folder on your Odin 3 (next to the Download folder)
